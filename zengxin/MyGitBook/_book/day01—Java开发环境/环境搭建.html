5. 字典配置

字典的作用：通过 字典的类型 与 字典的键 ，可以得到 字典的值

流程：

     graph LR
     A(首页)-->B(字典配置页面)-->C(新建字典类型)-->D(新建字典键与值)

7.1.1 新建字典类型

演示：

步骤：

  1. 在首页的右上角点击 设置
  2. 在设置的左下角点击 高级设置
  3. 在左方列表的 系统配置 中，找到 字典配置 并点击，即可来到字典配置的页面
  4. 在页面中点击 添加 字典类型
  5. 在字典类型设置中配置好参数后，点击保存，即可创建好一个字典类型，参数如下
    • 字典主 Code ：为字典类型的 字段名
    • 字典说明：为字典类型的 中文名

7.1.2 新建字典键与值

演示：

步骤：

  1. 在字典配置页面 选择一个字典类型 后，点击添加字典信息
  2. 在字典表设置中配置好参数后，点击保存，即可创建好一个字典的键值对，参数如下
    • 字典 Code ：字典键的 字段名
    • 字典说明：字典键的说明
    • 字典类型：字典 值的类型
    • 字典参数值：字典的值
