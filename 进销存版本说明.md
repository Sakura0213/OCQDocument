## 2. 基本流程

**1. 采购产品到仓库：**

```mermaid
graph LR
D(供应商)-->A(产品采购单)-->B(采购入库)
```



**2. 采购原材料加工成产品到仓库：**

```mermaid
graph LR
G(供应商)-->A(原材料采购单)-->B(采购入库)-->C(生产工单)-->D(领料出库)-->E(完工入库)-->K(工单结单)
```



**3. 客户下订单到产品出库：**

```mermaid
graph LR
D(客户)-->A(客户订单)-->B(销售出库)
```

**注意：**

1. 采购和销售会有退货的情况，可见下文
2. 若需要仓库的多仓位功能，则应该在系统的最开始就应该开启



## 3. 仓库

### 3.1 仓库与仓位

**介绍：**

1. 仓库：即存放产品库存的位置，公司中可以有多个仓库，如仓库A、仓库B等
2. 仓位：指的是在公司的仓库中，划分出的不同的仓库位置，如货架A1、货架A2等

<img src="https://files.catbox.moe/cl67sc.png#pic_center" alt="zengxin20230530164439" style="margin: 0px auto; zoom: 80%;" />



#### 3.1.1 新建仓库

**演示：**

<img src="https://files.catbox.moe/8jg1t1.png" alt="image-20230531110705455" style="zoom: 67%;" />

**步骤：**

1. 在仓库表右上角点击新建仓库（仓库表在：导航栏→仓库→仓库）
2. 填写仓库的信息
3. 点击保存，即可新建仓库



#### 3.1.2 开启多仓位

**演示：**

<img src="https://files.catbox.moe/cdhv8o.png" alt="zengxin20230530171954" style="zoom:77%;" />

**步骤：**

1. 在首页的右上角点击设置
2. 在设置的左下角点击高级设置
3. 在左方的条目中点击系统配置，再点击字典配置
4. 在字典类型中点击 “进销存”

**演示：**

<img src="https://files.catbox.moe/455kvs.png" alt="zengxin20230530173429" style="zoom:67%;" />

**步骤：**

1. 找到字典 MultiplePositions
2. 选中该字典，点击编辑
3. 将字典值参数改为1，保存后即为开启多仓位



#### 3.1.3 新建仓位

**演示：**

<img src="https://files.catbox.moe/14nqmb.png" alt="image-20230531111314764" style="zoom:67%;" />

**步骤：**

1. 在仓位表右上角点击新建仓位（仓位表在：导航栏→仓库→仓位）
2. 填写仓库的信息
3. 点击保存，即可新建仓位



### 3.2 产品库存

#### 3.2.1 新建产品

**演示：**

<img src="https://files.catbox.moe/f8jx4s.png" alt="image-20230531163318811" style="zoom: 67%;" />

**步骤：**

1. 在产品库存表右上角点击新建产品（产品库存表在：导航栏→仓库→产品库存）
1. 填入产品的基本信息后，选择产品存放的默认仓库
1. 启用批号：若需要该产品有批号的需求则在此处开启批号



#### 3.2.2 批号

**介绍：**即原材料或产品的保质期等属性会有不同，则物品进入到仓库时可设置批号，用来区分物品的不同批次。在使用原材料或出售产品时，可选择不同的批号。



### 3.3 产品库存表

**介绍：**产品库存表为产品库存中原材料和产品的详细信息，可根据产品名称、仓库、仓位、批号等信息来查询物品

![zengxin20230531174150](https://files.catbox.moe/f5d5ig.png)





### 3.4 库存异动记录

**介绍：**储存着所有对产品库存操作的详细信息，操作类型有采购入库、采购退货、领料出库、完工入库、销售出库和销售退货

**注意：**每次查看前需进行手动刷新，以获取最新的数据

![zengxin20230601101006](https://files.catbox.moe/o7h1cm.png)



### 3.5 库存盘点

**介绍：**线下盘点实际库存与系统中的库存会有差异，有可能线下拿货的情况没有录入到系统中，导致库存差异。则可在库存盘点中进行库存的变更。

**流程：**例如一个产品的库存少了10件

```mermaid
graph LR
A(在库存盘点中修改该产品实际库存)-->B(在产品库存表中矫正该产品的库存)
```

**演示：**

<img src="https://files.catbox.moe/7uxwho.png" alt="zengxin20230601142354" style="zoom:50%;" />

**步骤：**

1. 在库存盘点表右上角点击新建库存盘点（库存盘点表在：导航栏→仓库→库存盘点）
2. 填入产品的实际库存后点击保存

**演示：**

![zengxin20230601144749](https://files.catbox.moe/jeuvr1.png)

**步骤：**

1. 在产品库存表中选中刚才被盘点后的产品（产品库存表在：导航栏→仓库→产品库存表）
2. 点击一键矫正库存
3. 在弹出框中点击确定，即可矫正库存成功

（注意：在库存盘点之后，必须矫正库存，若不矫正库存，则该产品在产品库存中的总库存字段会出错）

## 4. 供应商

### 4.1 新建供应商

**系统路径：**【导航栏】➞【业务】➞【供应商】



**主要操作：**

**1.新建供应商**

* 演示：

  <img src="https://files.catbox.moe/m2lc5o.png" alt="QQ截图20230602153838" style="zoom:49%;" />

* 步骤：

  1. 在供应商页面右上角点击新建供应商。
  2. 填写供应商的公司名称、联系人等信息。
  3. 点击保存，即可新建供应商。



### 4.2 采购单

**介绍：**向供应商提供的采购单，可采购产品、原材料等。

**系统路径：**【导航栏】➞【业务】➞【采购单】

**部分字段说明：**

| 字段名称 | 说明                                 |
| -------- | ------------------------------------ |
| 已入库   | 在后续采购入库操作中，入库的产品数量 |
| 未入库   | 数量-已入库数量                      |



**主要操作：**

**1. 新建采购单**

* 演示：

  <img src="https://files.catbox.moe/s2w5tz.png" alt="QQ截图20230602155210" style="zoom:50%;" />

* 步骤：

  1. 在采购订单表右上角点击新建采购订单。
  2. 填写供应商等信息。
  3. 在明细中填写采购的产品。
  4. 点击保存，即可新建采购单。



### 4.3 采购入库

**介绍：**有采购单后，就可以进行实际的采购入库操作。设置产品不同的批号，就可以对同一张采购单上的产品进行多次入库操作。

**系统路径：**【导航栏】➞【仓库】➞【采购入库】

**部分字段说明：**

| 字段名称   | 说明                                         |
| ---------- | -------------------------------------------- |
| 仓位       | 若开启了多仓位，则需要填入具体仓库的具体仓位 |
| 批号       | 若开启了批号，则需要填入批号                 |
| 数量       | 为一个包装单位的数量                         |
| 单位换算率 | 不同的包装单位可以设置不同的单位换算率       |
| 基本数量   | 数量*单位换算率                              |
| 金额       | 基本数量*单价                                |



**主要操作：**

**1.新建采购入库**

* 演示：

  <img src="https://files.catbox.moe/yvzzjt.png" alt="image-20230601160642626" style="zoom: 46%;" />

* 步骤：

  1. 在入库记录页面右上角点击新建入库记录。
  2. 填写供应商、关联的采购单等信息。
  3. 在明细中填写采购入库的产品。
  4. 录入信息后点击保存，即可将采购的产品入库。





### 4.4 采购退货

**介绍：**在产品采购入库后，产品可能会出现质量等问题，则会产生采购退货。退货后，产品在库存中的数量将减少。

**系统路径：**【导航栏】➞【仓库】➞【采购退货】

**部分字段说明：**

| 字段名称   | 说明                                         |
| ---------- | -------------------------------------------- |
| 仓位       | 若开启了多仓位，则需要填入具体仓库的具体仓位 |
| 批号       | 若开启了批号，则需要填入批号                 |
| 数量       | 为一个包装单位的数量                         |
| 单位换算率 | 不同的包装单位可以设置不同的单位换算率       |
| 基本数量   | 数量*单位换算率                              |
| 金额       | 基本数量*单价                                |



**主要操作：**

**1. 新建采购退货**

* 演示：

  <img src="https://files.catbox.moe/gvzhdc.png" alt="image-20230602111740453" style="zoom: 50%;" />

* 步骤：

  1. 在采购退货表右上角点击新建采购退货。
  2. 填写供应商，关联采购单、入库记录等信息。
  3. 在明细中填写产品信息及退货的数量。
  4. 点击保存，即可新建采购退货。







## 5. 生产

### 5.1 生产工单

**介绍：**生产工单是向生产车间下达的生产计划单，记载产品、计划生产数量等生产信息内容。

**系统路径：**【导航栏】➞【生产】➞【生产工单】

**部分字段说明：**

| 字段名称 | 说明                                   |
| -------- | -------------------------------------- |
| 费用总额 | 除生产原料成本以外的费用               |
| 工单状态 | 无需修改，在主动结单后，状态会自动改变 |



**主要操作：**

**1. 新建生产工单**

* 演示：

  <img src="https://files.catbox.moe/eo3elu.png" alt="QQ截图20230602133852" style="zoom: 50%;" />

* 步骤：

  1. 在生产工单页面右上角点击新建生产工单。
  2. 填入产品、计划生产数量等信息。
  3. 点击保存后，即可成功新建生产工单。

**2. 主动结单**

* 介绍：在完成完成生产并入库后，即流程生产工单→领料出库→完工入库的流程后，生产出的产品并不会直接进入到库存中，需要进行主动结单操作后，产品才能被录入库存（注意：结单为生产的最后一步，且结单后工单将被锁定，所以完成工单上所有的生产计划后才能去结单）

* 演示：

  <img src="https://files.catbox.moe/i6z678.png" alt="QQ截图20230602142954" style="zoom: 61%;" />

* 步骤：

  1. 在生产工单页面中，勾选一条工单。
  1. 点击结单按钮。
  1. 点击确定结单，即可完成结单。
