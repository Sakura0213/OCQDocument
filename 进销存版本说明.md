## 2. 基本流程

**1. 采购产品到仓库：**

```mermaid
graph LR
D(供应商)-->A(产品采购单)-->B(采购入库)
```



**2. 采购原材料加工成产品到仓库：**

```mermaid
graph LR
G(供应商)-->A(原材料采购单)-->B(采购入库)-->C(生产工单)-->D(领料出库)-->E(完工入库)
```



**3. 客户下订单到产品出库：**

```mermaid
graph LR
D(客户)-->A(客户订单)-->B(销售出库)
```

**注意：**

1. 采购和销售会有退货的情况，可见下文
2. 若需要仓库的多仓位功能，则应该在系统的最开始就应该开启



## 3. 仓库

### 3.1 仓库与仓位

**介绍：**

1. 仓库：即存放产品库存的位置，公司中可以有多个仓库，如仓库A、仓库B等
2. 仓位：指的是在公司的仓库中，划分出的不同的仓库位置，如货架A1、货架A2等

<img src="https://files.catbox.moe/cl67sc.png#pic_center" alt="zengxin20230530164439" style="margin: 0px auto; zoom: 80%;" />



#### 3.1.1 新建仓库

**演示：**

<img src="https://files.catbox.moe/8jg1t1.png" alt="image-20230531110705455" style="zoom: 67%;" />

**步骤：**

1. 在仓库表右上角点击新建仓库（仓库表在：导航栏→仓库→仓库）
2. 填写仓库的信息
3. 点击保存，即可新建仓库



#### 3.1.2 开启多仓位

**演示：**

<img src="https://files.catbox.moe/cdhv8o.png" alt="zengxin20230530171954" style="zoom:77%;" />

**步骤：**

1. 在首页的右上角点击设置
2. 在设置的左下角点击高级设置
3. 在左方的条目中点击系统配置，再点击字典配置
4. 在字典类型中点击 “进销存”

**演示：**

<img src="https://files.catbox.moe/455kvs.png" alt="zengxin20230530173429" style="zoom:67%;" />

**步骤：**

1. 找到字典 MultiplePositions
2. 选中该字典，点击编辑
3. 将字典值参数改为1，保存后即为开启多仓位



#### 3.1.3 新建仓位

**演示：**

<img src="https://files.catbox.moe/14nqmb.png" alt="image-20230531111314764" style="zoom:67%;" />

**步骤：**

1. 在仓位表右上角点击新建仓位（仓位表在：导航栏→仓库→仓位）
2. 填写仓库的信息
3. 点击保存，即可新建仓位



### 3.2 产品库存

#### 3.2.1 新建产品

**演示：**

<img src="https://files.catbox.moe/f8jx4s.png" alt="image-20230531163318811" style="zoom: 67%;" />

**步骤：**

1. 在产品库存表右上角点击新建产品（产品库存表在：导航栏→仓库→产品库存）
1. 填入产品的基本信息后，选择产品存放的默认仓库
1. 启用批号：若需要该产品有批号的需求则在此处开启批号



#### 3.2.2 批号

**介绍：**即原材料或产品的保质期等属性会有不同，则物品进入到仓库时可设置批号，用来区分物品的不同批次。在使用原材料或出售产品时，可选择不同的批号。



### 3.3 产品库存表

**介绍：**产品库存表为产品库存中原材料和产品的详细信息，可根据产品名称、仓库、仓位、批号等信息来查询物品

![zengxin20230531174150](https://files.catbox.moe/f5d5ig.png)





### 3.4 库存异动记录

**介绍：**储存着所有对产品库存操作的详细信息，操作类型有采购入库、采购退货、领料出库、完工入库、销售出库和销售退货

**注意：**每次查看前需进行手动刷新，以获取最新的数据

![zengxin20230601101006](https://files.catbox.moe/o7h1cm.png)



### 3.5 库存盘点

**介绍：**线下盘点实际库存与系统中的库存会有差异，有可能线下拿货的情况没有录入到系统中，导致库存差异。则可在库存盘点中进行库存的变更。

**流程：**例如一个产品的库存少了10件

```mermaid
graph LR
A(在库存盘点中修改该产品实际库存)-->B(在产品库存表中矫正该产品的库存)
```

**演示：**

<img src="https://files.catbox.moe/7uxwho.png" alt="zengxin20230601142354" style="zoom:50%;" />

**步骤：**

1. 在库存盘点表右上角点击新建库存盘点（库存盘点表在：导航栏→仓库→库存盘点）
2. 填入产品的实际库存后点击保存

**演示：**

![zengxin20230601144749](https://files.catbox.moe/jeuvr1.png)

**步骤：**

1. 在产品库存表中选中刚才被盘点后的产品（产品库存表在：导航栏→仓库→产品库存表）
2. 点击一键矫正库存
3. 在弹出框中点击确定，即可矫正库存成功

（注意：在库存盘点之后，必须矫正库存，若不矫正库存，则该产品在产品库存中的总库存字段会出错）

**步骤：**

1. 在供应商表右上角点击新建供应商（供应商表在：导航栏→业务→供应商）
2. 填写供应商的信息
3. 点击保存，即可新建供应商



### 4.2 采购单

**演示：**

<img src="https://files.catbox.moe/use4qc.png" alt="image-20230601150406936" style="zoom:67%;" />

**步骤：**

1. 在采购订单表右上角点击新建采购订单（供应商表在：导航栏→业务→采购单）
2. 填写供应商、采购产品等信息后
3. 点击保存，即可新建采购单



### 4.3 采购入库

**介绍：**有采购单后，就可以进行实际的采购入库操作。设置产品不同的批号，就可以对同一张采购单上的产品进行多次入库操作。

<img src="https://files.catbox.moe/yvzzjt.png" alt="image-20230601160642626" style="zoom: 55%;" />

**步骤：**

1. 在入库记录表右上角点击新建入库记录（入库记录表在：导航栏→仓库→采购入库）
2. 输入供应商、关联的采购单等信息
3. 在关联采购单后，会自动填入采购单上产品的一些信息，明细的参数如下
   1. 仓位：若开启了多仓位，则需要填入具体仓库的具体仓位
   2. 批号：若开启了批号，则需要填入批号
   3. 数量：为一个包装单位的数量
   4. 单位换算率：不同的包装单位可以设置不同的单位换算率
   5. 基本数量：数量*单位换算率
   6. 金额：基本数量*单价
4. 录入信息后点击保存，即可将采购的产品入库





### 4.4 采购退货

**介绍：**在产品采购入库后，采购的产品可能会有质量等问题，则会产生退货，那么可以在采购退货中进行退货

**演示：**

<img src="https://files.catbox.moe/gvzhdc.png" alt="image-20230602111740453" style="zoom:67%;" />

**步骤：**

1. 在采购退货表右上角点击新建采购退货（采购退货表在：导航栏→仓库→采购退货）
2. 选择供应商，关联采购单、入库记录，填写相关信息
3. 在明细中填写产品信息及退货的数量
4. 点击保存，即可新建采购退货
