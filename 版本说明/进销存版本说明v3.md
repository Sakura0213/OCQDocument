## 4. 仓库



### 4.1 仓库

**功能介绍：**仓库是产品的存放地点，是产品进行库存管理的前提。

**应用场景：**

* 仓库的新增、删除、查询、编辑；
* 标识仓库的状态，如启用、停用。

**系统路径：**[ 导航栏 ] ➞ [ 仓库 ] ➞ [ 仓库 ]

**字段说明：**

| 字段 | 类型     | 说明                                                         |
| ---- | -------- | ------------------------------------------------------------ |
| NO   | 自动编号 | 含义：仓库的唯一编码&#xA;来源：系统生成                      |
| 名称 | 单行文本 | 含义：仓库的名称&#xA;来源：手工录入                          |
| 状态 | 下拉单选 | 含义：标识仓库是否启用&#xA;来源：手工录入&#xA;范围：默认为 [ 启用 ]，可选择 [ 启用 ] 或 [ 停用 ] |
| 位置 | 单行文本 | 含义：仓库的位置&#xA;来源：手工录入                          |
| 备注 | 多行文本 | 含义：仓库的相关备注&#xA;来源：手工录入                      |

**主要操作：**

   <span id="xjck">**1. 新建仓库**</span>

* 演示：

  <img src="https://files.catbox.moe/v3vhkd.png" alt="QQ截图20230605095530" style="zoom:48%;" />

* 步骤：

  1. 在仓库页面右上角点击 <font color="#F56600">**[ 新建仓库 ]**</font> 按钮。
  2. 填入仓库的名称、状态等信息。
  3. 保存后即可新建仓库。



### 4.2 仓位

**功能介绍：**仓位是仓库中不同的分类维度，可以是按仓库中的不同位置进行分类，如货架 A1 、货架 A2 等。也可以是按物料类型进行分类，如产品 I 类、产品 II 类等。（<font color=red>**注：**</font>若需要仓库的仓位功能，则应该在 <font color=red>**系统的最开始**</font> 就应该开启)

<img src="https://files.catbox.moe/h8h5rx.png" alt="QQ截图20230605142627" style="zoom:80%;" />

**应用场景：**

* 仓位的新增、删除、查询、编辑。

**系统路径：**[ 导航栏 ] ➞ [ 仓库 ] ➞ [ 仓位 ]

**字段说明：**

| 字段     | 类型     | 说明                                        |
| -------- | -------- | ------------------------------------------- |
| 名称     | 单行文本 | 含义：仓位的名称&#xA;来源：手工录入         |
| 所属仓库 | 引用     | 含义：仓位所在的仓库&#xA;来源：仓库表       |
| 位置     | 单行文本 | 含义：仓位在仓库中的位置&#xA;来源：手工录入 |
| 备注     | 多行文本 | 含义：仓位的相关备注&#xA;来源：手工录入     |

**主要操作：**

   <span id="kqcw">**1. 开启仓位**</span>

* 路径：[ 设置 ] ➞ [ 高级设置 ] ➞ [ 系统配置 ] ➞ [ 字典配置 ] ➞ [ 字典主类型 ]

* 演示：

  <img src="https://files.catbox.moe/455kvs.png" alt="zengxin20230530173429" style="zoom:58%;" />

* 步骤：

  1. 在字典配置页面的字典主类型中，点击 <font color="#F56600">**[ 进销存 ]**</font> 类型。
  2. 勾选字典 MultiplePositions。
  3. 点击 <font color="#F56600">**[ 编辑 ]**</font> 按钮。
  4. 将字典值参数改为 1 ，保存后即可开启多仓位。



   <span id="xjcw">**2. 新建仓位**</span>

* 步骤：

  1. 在仓位页面右上角点击 <font color="#F56600">**[ 新建仓位 ]**</font> 按钮。
  2. 填入仓位的名称、所属仓库等信息。
  3. 保存后即可新建仓位。



### 4.3 产品

**应用场景：**

* 产品的新增、删除、查询和编辑；
* 产品核算主体的批量修改；
* 产品状态更改。

**系统路径：**[ 导航栏 ] ➞ [ 仓库 ] ➞ [ 产品库存 ]

* 产品信息

  | 分类         | 分类数据    | 含义：产品的分类，在产品列表的左树，点击类别可对产品进行分类&#xA;来源：分类数据的产品分类&#xA;范围：可选择[ 成品类 ]、[ 材料类 ] 、[ 服务类 ] 或 [ 其他类 ] |
  | 产品编码     | 单行文本    | 含义：产品本身的编码&#xA;来源：手工录入                      |
  | 产品名称     | 单行文本    | 含义：产品的名称&#xA;来源：手工录入&#xA;范围：唯一，不可重复 |
  | 规格型号     | 单行文本    | 含义：产品的规格型号&#xA;来源：手工录入                      |
  | 状态         | 下拉单选    | 含义：产品的状态，在下订单等需要选择产品的操作中，禁用状态的产品将不可见&#xA;来源：手工录入&#xA;范围：默认为 [ 激活 ]，可选择 [ 激活 ] 或 [ 禁用 ] |

  | 默认仓库     | 引用        | 含义：产品的默认仓库，在采购入库操作中，选择产品后，自动将产品的默认仓库填入&#xA;来源：仓库的 |

  | 备注         | 多行文本    | 含义：产品备注&#xA;来源：手工录入                            |
  | 消耗数量     | 数字[小数]  | 含义：&#xA;来源：                                            |
  | 总货值       | 货币        | 含义：产品的总货值&#xA;来源：产品在产品库存表中库存金额的总和 |
  | 总库存       | 数字[小数]  | 含义：产品的总库存&#xA;来源：产品在产品库存表中库存数量的总和 |

  1. 在产品库存表右上角点击 <font color="#F56600">**[ 新建产品 ]**</font> 按钮。
  2. 填入产品的名称、售价、存放的仓库、是否启用批号等信息。

* 演示：

  <img src="https://files.catbox.moe/a37i7v.png" alt="image-20230606151907177" style="zoom:74%;" />

  1. 在字典配置页面的字典主类型中，点击 <font color="#F56600">**[ 进销存 ]**</font> 条目。

  3. 点击 <font color="#F56600">**[ 编辑 ]**</font> 按钮。
  4. 将字典值参数改为 1 ，保存后即可开启负库存。



### 4.4 产品库存表

**功能介绍：**产品库存表为产品库存中产品的详细信息，记载产品的具体仓库、仓位、批号等信息，可根据这些信息查询产品。

**应用场景：**

* 产品库存的查询；
* 产品库存的一键矫正。

**系统路径：**[ 导航栏 ] ➞ [ 仓库 ] ➞ [ 产品库存表 ]

**字段说明：**

* 产品信息

  | 字段     | 类型       | 说明                                               |
  | -------- | ---------- | -------------------------------------------------- |
  | 产品名称 | 引用       | 含义：产品的名称                                   |
  | 产品编号 | 单行文本   | 含义：产品的编号                                   |
  | 基本单位 | 引用       | 含义：产品的基本单位                               |
  | 仓库     | 引用       | 含义：产品的所在仓库                               |
  | 仓位     | 引用       | 含义：产品的所在仓位                               |
  | 批号     | 单行文本   | 含义：产品的批号                                   |
  | 生产日期 | 日期时间   | 含义：产品的生产日期                               |
  | 保质期   | 数字[整数] | 含义：产品的保质期                                 |

  | 库存数量 | 数字[小数] | 含义：产品的库存数量                               |
  | 库存金额 | 数字[整数] | 含义：产品的库存金额                               |
  | 平均单价 | 数字[货币] | 含义：产品的平均单价&#xA;来源：库存金额 / 库存数量 |

  <img src="https://files.catbox.moe/vdxtnj.png" alt="QQ截图20230605102738" style="zoom:58%;" />

  2. 点击 <font color="#F56600">**[ 一键校正库存 ]**</font> 按钮后即可校正库存。

  | 仓库     | 引用       | 含义：产品所在的仓库                       |
  | 仓位     | 引用       | 含义：产品所在的仓位                       |
  | 批号     | 文本类型   | 含义：产品的批号                           |

  | 出库数量 | 数字[小数] | 含义：本次操作中，从产品库存表中减少的数量 |
  | 库存数量 | 数字[小数] | 含义：产品的库存数量&#xA;来源：产品库存表  |
  | 库存金额 | 数字[货币] | 含义：产品的库存金额&#xA;来源：产品库存表  |

### 4.6 库存盘点

**功能介绍：**线下进行实际库存的盘点时，实际库存可能与系统库存会存在差异，有可能是线下拿货的情况没有录入到系统中，导致库存差异。则可以在库存盘点中进行库存的修改。
