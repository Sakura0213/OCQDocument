### 4.1 储存仓库

**<span style="font-size: 19px;">功能介绍</span>**

该模块是系统初始阶段需要准备的数据，用于管理和跟踪企业物料和产品的存放位置。通过储存仓库模块，企业可以设定和管理不同的仓库，并记录仓库的名称、位置、状态等信息。



**<span style="font-size: 19px;">应用场景</span>**

* 仓库的新增、删除、查询、修改；
* 标识仓库的启用状态，当某个仓库停止使用时，不能删除该仓库，而是应将其状态修改为 < 停用 >。



**<span style="font-size: 19px;">系统路径</span>**

[ 导航栏 ] ➞ [ 仓库 ] ➞ [ 仓库 ]



**<span style="font-size: 19px;">字段说明</span>**

| 字段 | 说明                                                         |
| ---- | ------------------------------------------------------------ |
| NO   | 含义：仓库的唯一标识，用于区分不同的仓库，方便在系统中进行查找和管理。&#xA;来源：系统生成 |
| 名称 | 含义：仓库的名称，通常是一个描述性的名称，便于用户快速识别和区分不同的仓库。&#xA;来源：手工录入 |
| 状态 | 含义：用于标识仓库的当前状态，有 < 启用 > 和 < 停用 >。状态字段可以帮助用户了解仓库的使用情况，以便做出相应的决策。另外，不能删除已经被使用的仓库，只能将其状态修改为 < 停用 > 。&#xA;来源：手工录入 |
| 位置 | 含义：记录仓库的具体位置信息，例如楼层、房间号等。这个字段可以帮助用户快速定位仓库的位置，方便管理和操作。&#xA;来源：手工录入 |
| 备注 | 含义：用户可以在这个字段中输入额外的备注信息，例如仓库的特殊要求、注意事项等。这个字段可以用于存储一些额外的细节信息，提供更全面的仓库信息。&#xA;来源：手工录入 |

 

**<span style="font-size: 19px;">主要操作</span>**

  * 保存后即可新建仓库。



 <span id="bsckzt">**2. 标识仓库状态**</span>

## 9. 其他操作

<span id="sjddr">**1. 数据的导入**</span>

> <font color="#F56600">**注：**</font>以仓库模块为例

* 演示：

  <img src="https://s3.bmp.ovh/imgs/2023/06/26/4492b5d88cdc07ea.png" alt="QQ截图20230626104428" style="zoom:50%;" />

* 步骤：

  * 点击仓库页面右上角下拉框中的 <font color="#F56600">**[ 导入 ]**</font> 按钮。

  * 选择一个 Excel 格式的文件。

  * 字段映射：左边为系统中的字段，右边为 Excel 的列字段，在这里需要将 Excel 的字段与系统字段对应

  * 点击开始导入后即可导入。

    

<span id="drscc">**2. 导入时查重：**</span>

  * 演示：

    <img src="https://s3.bmp.ovh/imgs/2023/06/26/c10ad6b649109792.png" alt="image-20230626105242409" style="zoom:67%;" />

  * 步骤：

    * 设置查重的字段。
    * 设置字段重复时的操作，入这里选择 < 不导入 >。
    * 则当仓库名称在系统中重复时，则重复的记录不会被导入，并会提示导入失败。



<span id="sjddc">**3. 数据的导出**</span>

> <font color="#F56600">**注：**</font>以仓库模块为例

* 演示：

  <img src="https://s3.bmp.ovh/imgs/2023/06/26/98d15fab270c2c37.png" alt="image-20230626102035938" style="zoom: 60%;" />

* 步骤：

  * 在仓库表中，勾选一条或多条记录。
  * 在右上角的下拉框中，点击 <font color="#F56600">**[ 导出 ]**</font> 按钮。
  * 选择导出的条件后点击确定，即可导出格式为 Excel 的仓库表。
